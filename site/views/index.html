debugging... with friends!

Stop emailing logs or have someone looking over your shoulder when you hit a bug! With nodbg, you can share your debugger with friends, even outside your own network! No logins or accounts, just start it up and go!

Install and Run

Try Now!

Install from npm

> npm install -g nodbg

Run locally

> nodbg local path/to/file_to_debug.js
- nodbg params: {}
- nodbg started!
- Connect to http://localhost:3000/debug/debug0

Run with nodbg.com

> nodbg path/to/file_to_debug.js
- nodbg params: {}
- nodbg started!
- Connect to http://nodbg.com/debug/debug0

You can try out nodbg.com without installing anything! We will launch a test program for you that you can debug and share with friends!
